## 适合谁读

- 正在准备校招/社招的工程师
- 想从小厂跳大厂但不知道怎么准备的人
- 简历投了很多但面试机会很少的人
- 面试通过率低、想系统提升面试能力的人

## 关于作者

在职大厂工程师，从实习、校招到社招，累计面试上百场，校招多个大厂 offer，近期社招通过率超过 90%。从本科开始辅导学弟学妹，毕业后陆续辅导了数十位不同背景的同学，覆盖暑期实习、秋招春招、社招跳槽、技术转型、发展路线规划。
